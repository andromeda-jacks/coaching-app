AI 기반 연구보고서 자동 생성 & 200점 자동 심사 시스템

## 배포

Vercel에서 바로 배포:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_USER/coaching-app)

## 로컬 실행

```bash
npm install
npm run dev
```

## 주요 기능

- 보고서 6개 섹션 AI 자동 생성
- 공식 심사기준(200점) 기반 자동 채점
- 개선 ROI 순위 → 수정 → 재심사 루프
- 참고자료(우수작 등) 컨텍스트 투입
- 18개 AI 서비스 지원 (Claude, Gemini, Groq 등)
- API 없이도 프롬프트 복사 모드로 100% 동작
- 모바일 완전 대응

## 기술 스택

React 18 · Vite · lucide-react · localStorage
